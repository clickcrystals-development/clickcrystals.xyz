Script Command: If

Evaluates an if statement.

*Notice

  • Argument format may not be uniform, some may require you do provide another one. A [**] prefix would indicate the argument is only needed for some specific previous arguments.
  • [..] argument would indicate any CCS script block or line can follow after.
  • [N] argument indicates any decimal number.
  • [n] argument indicates any whole number number (integer).
  • [+N] argument indicates an operater in front of a decimal number, there are NO SPACES between the two.
  • [+n] argument indicates an operater in front of whole number (integer), there are NO SPACES between the two.
  • [ID] argument indicates minecraft identifiers. Prefix [:] implies exact match, prefix [#] implies match contains.

Arguments

  1. condition
  2. **specifier
  3. **specifier_value
