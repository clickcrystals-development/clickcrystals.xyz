Syntax

Script Command Description Documentation
on Registers an event listener. link
if Evaluates an if statement. link
if_not Evaluates an if statement and tests if value is false. link
while Loops a CCS block or line until the condition is false. link
while_not Loops a CCS block or line until the condition is true. link
execute Executes any CCS script block or line. link
execute_random Executes ONE RANDOM CCS script line from the next script block. link
loop Repeats a CCS block or line for the specified amount of times. link
loop_period Repeats a CCS block or line for the specified amount of times, while also waiting a period before the next iteration. link
print Prints the next quoted message in the console. link
throw Throws an exception (error) with the message from the next quoted message. link
exit Exits the Java JVM with the specified exit code. link
function Calls a script function with the specified name. link

Client

Script Command Description Documentation
module Manages modules with the specified module index. link
description Sets the description of the current script module to the next quoted message. link
config Manages the config for ClickCrystals. link
say Makes the client type or execute the message of the next quote in chat. link
send Direct messages the session user with the message of the next quote. link
notify Sends a notification to the session user with the specified stay time and the message of the next quoted message. link
playsound Plays a sound to the client session user with the specified volume and pitch. link
define Defines a specific object for the client. Mostly used for scripting. link

Macros

Script Command Description Documentation
drop Drops the item you have in your mainhand with the specified amount. link
teleport Sends a teleport packet to the server to change your position instantly. link
turn_to Slowly turns your head camera to a specified object in the world to simulate player input. link
snap_to Snaps your head camera instantly to a specified object in the world. Does not simulate player input. link
damage Sends a attack packet to the nearest of the specified entity. link
switch Hotkeys to the specified item in your hotbar. link
swap Swaps the item in your main hand with the one in your offhand. link
input Simulates a player input: attack, use, walk, etc... link
gui_drop If inventory is opened, drop the specificed item with the specified amount. link
gui_swap If inventory is opened, swap the specified item with your offhand item. link
gui_switch If inventory is opened, hover cursor over the specified item. link
gui_quickmove If inventory is opened, quickmove the specified item. link
