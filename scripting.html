What is it?

ClickCrystals Script is an in-game module maker that allows you to create custom features. Have an idea but it never got accepted? Script it!

Easy syntax with NO variables to worry about! Just script!

Think of this as minecraft commands.

Creating a module

Tutorial:

module create (module-name)
description (Text)

Example:

module create new-module
description This is describing the new-module

Before we continue

Here's the key legend to assist you while you read the documentations below:

  • () surrounds argument types
  • leading .. means argument only sometimes appear before
  • trailing .. means argument only sometimes appear after
  • (CCS.. args) means more script command lines can follow after; execute whatever you want
  • (N) means any number
  • (+N) doesn't mean positive numbers, it means expressioned number. For example >=0.9
  • ("") means a string in quotes. when writing strings in script, always surround then in quotation marks!
  • (ID) represents a minecraft ID value. # prefix means ID name contains the prefixed phrase, and : means ID name EQUALS to the prefixed phrase.

Example 1:

module create new-module
description This is describing the new-module

on key_press space {
   if block_in_range #diamond_ore 16 {
      send "Diamond within 16 blocks!"
      turn_to nearest_block #diamond_ore
   }
   send "you pressed space bar"
}

Example 2:

module create new-module
description This is describing the new-module

on totem_pop {
   if hotbar_has #totem {
      switch #totem

      wait 0.05 {
        swap
        switch back
      }
   }
   send "you popped!"
}

Commands:

Syntax:

  • on (event..) (..value) (CCS.. args)
  • if (condition..) (..expression..) (..value) (CCS.. args)
  • if_not (condition..) (..expression..) (..value) (CCS.. args)
  • while (condition..) (..expression..) (..value) (CCS.. args)
  • while_not (condition..) (..expression..) (..value) (CCS.. args)
  • execute (CCS.. args)
  • execute_random (CCS.. args)
  • loop (N) (CCS.. args)
  • loop_period (N) (N) (CCS.. args)
  • print ("")
  • throw ("")
  • exit (N)

Client:

  • module (action) (value)
  • description ("")
  • config (action)
  • say ("")
  • send ("")
  • notify ("")
  • playsound (ID..) (..N..) (..N)
