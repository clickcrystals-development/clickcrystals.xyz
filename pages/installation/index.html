Fabric (ClickCrystals)

  • Download the Fabric Loader
  • Download the Fabric API
  • Download the Fabric version of the mod that you want to install
  • Execute the Fabric Installer
  • Open the .minecraft folder:
    • Windows: use Windows+R and type %appdata%, then go into the .minecraft/ folder
    • Linux: open your Home folder and go to .minecraft/ (the folder is hidden)
  • Open the mods/ folder and move there the .jar file
  • Launch the Fabric installation from the Laucher

Texture Pack (ClickCrystals Texture Packs)

  • Download the texture pack from downloads that you want to install
  • Open the .minecraft folder:
    • Windows: use Ctrl+R and type %appdata%, then go into the .minecraft/ folder
    • Linux: open your Home folder and go to .minecraft/ (the folder is hidden)
  • Open the resourcepack/ folder and move there the .zip files
  • Launch your Minecraft installation from the Launcher
  • Open Options > Resource Packs
  • Enable the resource pack that you installed
